# ZYNQ7020_coreboard
基于ZYNQ7020的fpga核心板
项目内容：通过立创eda设计了一块六层fpga开发板，配备一颗4GbDDR3、一颗flash、一路千兆网口、一路SD卡、32对差分IO。独立完成电路与pcb设计，并成功打板焊接验证开发板功能。

电路设计：根据主控bank电压确定ddr、phy型号并设计外围电路，配置启动状态。选用tps82130dcdc模块作为板级电源，通过PG与EN引脚级联控制启动时序。配备拨码开关切换不同启动模式。

PCB设计：使用SGSSPS叠层，根据手册要求阻抗分别计算DDR与网口的线宽，并分配走线同组同层，等长误差3mil。电源层采用分割铺铜，保证载流能力与相邻层参考平面完整。

焊接测试：完成了FPGA、DDR与其他器件的焊接。经测试上电顺序与电源纹波均符合要求，可使用vivado正常烧录PL端程序。使用VITIS进行DDR读写与网口收发测试，性能均能达标。
